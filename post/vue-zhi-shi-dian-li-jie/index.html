模板字符串的前世今生

JS

js中，输出模板是这样写的。通过字符串拼接

$('#result').append(
 'There are <b>' + basket.count + '</b> ' +
 'items in your basket, ' +
 '<em>' + basket.onSale + '</em> are on sale!'
);

es6

es6中，输出模板是这样写的。通过${ }嵌入变量

$('#result').append(`
    There are <b>${basket.count}</b> items
    in your basket, <em>${basket.onSale}</em>
    are on sale!
`);

Vue

Vue中，输出模板是这样写的。

new Vue({
     el:'#replace',
    template:'<p>字符串模板</p>'
})

Vue2.0模板的三种写法

第一种：利用模板字符串

<script type="text/javascript">
    var app=new Vue({
        el:"#app",
        data:{ },
        template:`<h1 style="color:red">第一种写法</h1>`
    })
</script>

第二种：利用template标签实现内容填充,通过 id 或者其他的选择器挂载到template上

通过 id,挂载到template上

<template id="template">
    <h1>11</h1>
</template>
<script type="text/javascript">
    var app=new Vue({
    ...
    template:"#template"
})
</script>

改变script中的type.变为type="x-template",给它id,最终挂载到template中

使用于内容比较多或有公共的模板,可以使用,引入就可以
type="x-template"，给它id

<script src="js/vue.js"></script>
<script type="x-template" id="template">
    <h2>11</h2>
</script>
<script type="text/javascript">
var app=new Vue({
     ...
    template:"#template"
})

Vue3.0模板的写法

import Vue from 'vue'
import App from './App.vue'
new Vue({
   ...
    render: h => h(App)
}).$mount('#app')
