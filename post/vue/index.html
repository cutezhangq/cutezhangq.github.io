路由动画

💥出现问题：动画滑动时，最上面的header元素没有置顶，导致最后动画完成时突然向上移动。
✔️解决：

<transition :name="transitionName">
    <router-view :Event_Play="Event_Play" class="transitionBody"><router-view>
</transition>
<style lang="less">
.transitionBody {
        position: absolute;
        left: 0;
        top: 0;
        width: 100%;
        height: 100%;
 }
 </style>
